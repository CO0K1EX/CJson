# CJson

CJSON is a CLI tool written in C++ that retrieves JSON responses from web APIs.


![CLI Example](https://user-images.githubusercontent.com/74844338/190232467-12862dc6-7189-4c5f-be9f-67e8271aa7e2.png)

# Features
 - Fetch JSON data from APIs
 - Simple command line interface
 - Uses Boost networking libraries

# Example

cjson https://ip-api.com/json

![CLI Result](https://user-images.githubusercontent.com/74844338/190232497-9b2ce1eb-7d72-4f42-b340-7cda815ce7f9.png)

## Technologies
 - C++
 - Boost
 - HTTP requests
 - JSON APIs
